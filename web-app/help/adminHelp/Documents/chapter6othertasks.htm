Chapter 6:  Other Tasks

In addition to Search, Dataset Explorer, and Gene Signature/List, the tranSMART toolbar includes the following tool tabs:

■    Help

Display links to the tranSMART online and PDF documentation sets.

■    Admin

Perform user administration functions such as creating users, assigning roles to users, and enabling or disabling user accounts.

This tool tab is visible only to users who are assigned the role ROLE_ADMIN.
